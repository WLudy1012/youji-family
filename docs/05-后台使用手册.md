# 由基家族网站系统 - 后台使用手册

## 一、登录后台

1. 访问管理后台：`http://your-domain.com/admin`
2. 输入默认账号密码：
   - 用户名：`admin`
   - 密码：`admin123`
3. 点击"登录"

**⚠️ 重要提示：首次登录后请立即修改默认密码！**

## 二、控制台概览

登录后进入控制台页面，可以看到：
- 家族成员数量
- 文章数量
- 相册数量
- 留言数量

## 三、成员管理

### 添加家族成员

1. 点击左侧菜单"成员管理"
2. 点击"新建"按钮
3. 填写成员信息：
   - **姓名**（必填）
   - **头像**：点击上传图片
   - **出生日期**：选择日期
   - **个人简介**：填写成员介绍
   - **辈分/代数**：选择第几代
   - **性别**：男/女
   - **联系方式**：电话、邮箱、地址
4. 点击"确定"保存

### 设置家族关系

在添加/编辑成员时，可以设置：
- **父母**：选择该成员的父母
- **配偶**：选择该成员的配偶

设置关系后，族谱页面会自动显示家族树。

### 为成员创建登录账号

1. 在成员列表中找到目标成员
2. 点击"创建账号"按钮
3. 设置：
   - 登录用户名
   - 密码
   - 邮箱（可选）
4. 点击"确定"

成员可以使用此账号登录查看专属内容。

### 编辑成员信息

1. 在成员列表中找到目标成员
2. 点击"编辑"按钮
3. 修改相关信息
4. 点击"确定"保存

### 删除成员

1. 在成员列表中找到目标成员
2. 点击"删除"按钮
3. 确认删除

⚠️ 删除成员会同时删除其关联的关系和账号！

## 四、文章管理

### 发布文章

1. 点击左侧菜单"文章管理"
2. 点击"新建"按钮
3. 填写文章信息：
   - **标题**（必填）
   - **分类**：选择或输入分类
   - **封面图**：上传封面图片
   - **摘要**：文章简介
   - **内容**：文章内容（支持富文本编辑）
   - **发布状态**：草稿/已发布
4. 点击"确定"保存

### 编辑文章

1. 在文章列表中找到目标文章
2. 点击"编辑"按钮
3. 修改内容
4. 点击"确定"保存

### 删除文章

1. 在文章列表中找到目标文章
2. 点击"删除"按钮
3. 确认删除

## 五、公告管理

### 发布公告

1. 点击左侧菜单"公告管理"
2. 点击"新建"按钮
3. 填写公告信息：
   - **标题**（必填）
   - **内容**（必填）
   - **是否置顶**：置顶公告会显示在最前面
   - **发布状态**：草稿/已发布
4. 点击"确定"保存

### 编辑公告

与文章编辑操作相同。

## 六、相册管理

### 创建相册

1. 点击左侧菜单"相册管理"
2. 点击"新建"按钮
3. 填写相册信息：
   - **相册名称**（必填）
   - **相册描述**
   - **封面图**：上传封面
   - **是否公开**
4. 点击"确定"保存

### 上传图片

1. 在相册列表中点击相册名称进入详情
2. 点击"上传图片"按钮
3. 选择要上传的图片（可多选）
4. 等待上传完成
5. 为每张图片添加说明（可选）

### 删除图片

1. 在相册详情中找到要删除的图片
2. 点击"删除"按钮
3. 确认删除

## 七、留言管理

### 审核留言

1. 点击左侧菜单"留言管理"
2. 查看待审核的留言
3. 点击"通过"或"拒绝"按钮

### 回复留言

1. 在留言列表中找到要回复的留言
2. 点击"回复"按钮
3. 输入回复内容
4. 点击"确定"

### 删除留言

1. 在留言列表中找到要删除的留言
2. 点击"删除"按钮
3. 确认删除

## 八、系统设置

### 修改站点信息

1. 点击左侧菜单"系统设置"
2. 修改以下信息：
   - **站点名称**：显示在网站标题和页脚
   - **站点LOGO**：上传网站LOGO
   - **站点描述**：用于SEO
   - **联系邮箱**：显示在页脚
   - **联系电话**：显示在页脚
   - **ICP备案号**：显示在页脚
   - **页脚版权信息**

3. 点击"保存"按钮

### 修改首页横幅

在系统设置中修改：
- **首页横幅标题**
- **首页横幅副标题**
- **家族宣言**

### 修改主题颜色

在系统设置中可以修改：
- **主题主色**：默认深蓝 (#1e3a5f)
- **主题辅色**：默认金色 (#c9a227)

修改后前台页面会自动应用新颜色。

## 九、修改管理员密码

1. 点击右上角用户名
2. 选择"修改密码"
3. 输入旧密码和新密码
4. 点击"确定"

## 十、退出登录

1. 点击右上角用户名
2. 选择"退出登录"

## 十一、常见问题

### Q: 如何修改"由基家族"这个名称？

A: 进入"系统设置"页面，修改"站点名称"字段，保存后即可生效。

### Q: 如何更换网站LOGO？

A: 进入"系统设置"页面，点击"站点LOGO"旁边的上传按钮，选择新的LOGO图片上传。

### Q: 如何修改首页显示的内容？

A: 首页显示的内容来自各个模块：
- 公告：在"公告管理"中添加
- 成员：在"成员管理"中添加
- 文章：在"文章管理"中添加
- 相册：在"相册管理"中添加

### Q: 成员账号和普通管理员账号有什么区别？

A: 
- **管理员账号**：可以管理整个网站的所有内容
- **成员账号**：只能查看自己的信息和部分专属内容

### Q: 如何备份网站数据？

A: 数据备份包括：
1. **数据库**：使用MySQL导出功能
2. **上传文件**：备份 `uploads` 目录

建议设置自动备份脚本，定期备份数据。

### Q: 网站访问很慢怎么办？

A: 可以尝试以下优化：
1. 启用CDN加速静态资源
2. 优化图片大小
3. 启用Gzip压缩（已默认开启）
4. 升级服务器配置
